Rust Package

cargo new hello 实际上是创建了一个叫hello的包.
crates 是一个模块的树形结构，它形成了库或二进制项目。crate 根文件将由 Cargo 传递给 rustc 来实际构建库或者二进制项目

如何判断是二进制或是库？
Cargo 规定包含： src/main.rs 的判断是二进制crates，包含src/lib.rs 的判断是库crates
