    1. 安装

    • Linux
      自带
    • Windows
      需要下载sqlite-tools-win32-.zip 和 sqlite-dll-win32-.zip（或者64那个版本），然后将这两个解压到同一个目录中比如 C:\sqlite 然后将这个路径添加到PATH环境变量中

    2. 命令

    在命令行输入 sqlite3 进入SQLite提示符
